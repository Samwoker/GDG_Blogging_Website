Welcome to the API Documentation

Here you can find all the available API endpoints for your application.

Base URL

https://gdg-blogging-website.onrender.com

Authentication Endpoints

POST /api/auth/register

Registers a new user.

Request Body:

    {
      "username": "string",  
      "email": "string",     
      "password": "string"   
    }

Response:

    {
      "user": {
        "_id": "user_id",
        "username": "string",
        "email": "string",
        "password": "hashed_password",
        "role": "user",
        "createdAt": "timestamp",
        "updatedAt": "timestamp",
        "__v": 0
      },
      "token": "jwt_token"
    }

POST /api/auth/login

Logs in an existing user and returns a JWT token.

Request Body:

    {
      "email": "string",     
      "password": "string"   
    }

Response:

    {
      "user": {
        "_id": "user_id",
        "username": "string",
        "email": "string",
        "password": "hashed_password",
        "role": "user",
        "createdAt": "timestamp",
        "updatedAt": "timestamp",
        "__v": 0
      },
      "token": "jwt_token"
    }

POST /api/auth/logout

Logs out the user by clearing the session or invalidating the token.

Response:

    {
      "message": "Logged out successfully"
    }

POST /api/auth/reset-password/request

Requests a password reset for a user by sending a reset email.

Request Body:

    {
      "email": "string"  
    }

Response:

    {
      "message": "Password reset link sent"
    }

POST /api/auth/reset-password/:token

Resets the user's password using the reset token.

Request Body:

    {
      "newPassword": "string"  
    }

Response:

    {
      "message": "Password has been reset successfully"
    }

GET /api/auth/google

Redirects to Google for authentication.

Response:

    {
      "message": "Redirecting to Google for authentication"
    }

GET /api/auth/google/callback

Handles the callback from Google and completes authentication.

Response:

    {
      "message": "Authentication successful, user logged in"
    }
